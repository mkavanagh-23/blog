## My Hugo Blog

A personal tech blog built and auto-deployed using hugo. For a full setup guide, look [here](https://matthewkavanagh.us/posts/2026-03-23_hugo-automation/)

### Setup
1. Clone this repo:
    
    ```bash
    git clone --recurse-submodules git@github.com:mkavanagh-23/blog.git
    ```

2. Push new markdown file to automate build and deployment:

    ```bash
    git add .
    git commit
    git push
    ```
