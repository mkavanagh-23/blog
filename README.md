## My Hugo Blog

A personal tech blog built and auto-deployed using hugo. For a full setup guide, look [here](https://matthewkavanagh.us/posts/2026-03-23_hugo-automation/)

### Setup
1. Clone this repo:
    
    ```bash
    git clone git@github.com:mkavanagh-23/blog.git
    ```

2. Initialize the theme submodule:

    ```bash
    git submodule update --init --recursive
    ```

3. Push new markdown file to automate build and deployment:

    ```bash
    git add .
    git commit
    git push
    ```
